This is the first post. It exists to verify that the blog infrastructure works:

## Typography

The body is set in Georgia at 18px with a line-height of 1.75. The goal is a reading
experience close to a well-typeset book — unhurried, uncluttered.

> Good typography is invisible. Bad typography gets in the way.

Headings switch to the system sans-serif to create a clean contrast. Links are
underlined with a single accent colour.

## Code

Here is some Python:

```python
def sieve_of_eratosthenes(n: int) -> list[int]:
    """Return all primes up to n."""
    is_prime = [True] * (n + 1)
    is_prime[0] = is_prime[1] = False
    for i in range(2, int(n**0.5) + 1):
        if is_prime[i]:
            for j in range(i * i, n + 1, i):
                is_prime[j] = False
    return [i for i, p in enumerate(is_prime) if p]
```
